## Brand Voice Rules

**NEVER Rules:**
- NEVER use cluttered or visually busy creative — brand is clean and minimalist
- NEVER reference cached/outdated data or make unreliable claims — transparency is a core pillar
- NEVER use generic benefit statements — specific pain-point framing converts dramatically better
- NEVER name competitors directly in ads — use "generic competitor" framing only
- NEVER imply complex setup or account creation — instant no-signup access is the key differentiator
- NEVER position Points Path as "just another points hack blog" — it is a decision-making tool
- NEVER create pure newsletter signup hooks without teeing up the extension install downstream

**Approved Voice:** Conversational, relatable, traveler-first. Direct address with problem identification: "If you have points you never use, you need to hear this." Loss aversion hooks dominate: "wasting points," "never use," "wrong way," "leaving money on the table." Decision-fatigue framing: "Points or cash? Just use Points Path." Authority/proof signals: "Redeeming through the credit card portal often gives you just 1 cent per point — vs. 6-12x through Points Path." No corporate or financial jargon.

**Winning Formats:** TOV (Text-Over-Video — dominant), Pain-Point Statics, UGC Video (emerging), Lead Magnet Funnel (emerging)
