## 3. BRAND VOICE RULES

### NEVER Rules
1. NEVER use "Serious money" / "Make money online" / hyperbolic income claims
2. NEVER use guru/hustle culture framing
3. NEVER use meme/shock/clickbait creative formats
4. NEVER include factual errors — wrong subscriber counts, improper capitalization, gendered language
5. NEVER show recognizable faces in video thumbnails without blurring

### Tone Aspiration
**The New Yorker / NYT classiness — "Real Journalism"**
